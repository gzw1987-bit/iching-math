听见64卦：二进制变成音乐

每个卦有6个爻（bit），每个bit对应一个音符。64卦×6音符=384个音符的旋律。
不同的排列方式产出完全不同的"曲子"——用耳朵识别眼睛看不到的模式。

模式一：旋律 — 6爻 → 6音阶

阴(0)=低音，阳(1)=高音。6个爻位映射到中国五声音阶+一个八度音的6个音高。每个卦变成一段6音符的短旋律。
初爻 0 宫 C4 262Hz
初爻 1 商 D4 294Hz
二爻 0 角 E4 330Hz
二爻 1 徵 G4 392Hz
三爻 0 羽 A4 440Hz
三爻 1 宫 C5 523Hz
四爻 0 商 D5 587Hz
四爻 1 角 E5 659Hz
五爻 0 徵 G5 784Hz
五爻 1 羽 A5 880Hz
上爻 0 宫 C6 1047Hz
上爻 1 商 D6 1175Hz
200 BPM
听的时候注意：自然序听起来像一段规律的、逐渐上升的音阶练习。文王序听起来像一段充满跳跃和对比的即兴演奏。格雷码极其平滑——因为每步只变一个音。你能听到文王的"叙事张力"吗？

模式二：和弦 — 每卦6个音同时响

不是依次播放6个音——而是6个音同时响。每个卦变成一个和弦。阳爻多的卦听起来"亮"（高频多），阴爻多的听起来"暗"（低频多）。坤=全低音和弦，乾=全高音和弦。
80 BPM
听的时候注意：按阳爻递增排列时，你会听到从"最暗"到"最亮"的渐变——一段从坤到乾的光谱。文王序的前两个和弦——乾(全亮)然后坤(全暗)——是全曲最大的对比，开篇就给你最强的冲击。

模式三：节奏 — 阳=击鼓，阴=静音

不是音高——是节奏。阳(1)=一声鼓点，阴(0)=静默。每个卦是一小节6拍的节奏型。乾=全击（XXXXXX），坤=全静（......），其他卦是各种切分节奏。
360 BPM
听的时候注意：自然序的节奏从极稀疏（坤附近）到极密集（乾附近）——像一个加速的心跳。文王序的节奏跳来跳去——密集和稀疏交替出现——这就是文王叙事的"呼吸节奏"。
