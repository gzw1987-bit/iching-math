两者的底层结构完全一样：在一组约束条件下，系统会稳定在哪里？

深层同构：为什么博弈论和易经说的是同一件事

博弈论的核心概念

参与者（Players）：博弈的主体

易经的对应

上卦/下卦：外部力量 vs 内部力量。每个卦本身就是一个两方博弈。

策略空间（Strategy Space）

每个参与者可选择的行动集合

每爻的阴阳选择

6个位置×2种选择 = 2⁶ = 64种策略组合 = 64卦

纳什均衡（Nash Equilibrium）

没有参与者有动力单方面改变策略的稳定状态

卦象

每一卦就是一个稳定态——在这种力量配置下，系统会停在这里

均衡转移

外部冲击打破旧均衡，系统跳到新均衡

变爻 → 之卦

某一爻到极限（老阳/老阴）翻转 → 卦象变化 = 从旧均衡跳到新均衡

机制设计（反向博弈论）

不是在规则下找最优解，而是设计规则使均衡行为达到目标

鼎/节/井

文王/周公做的就是机制设计——设计礼乐制度使"走完10卦循环"成为均衡行为

重复博弈

同一博弈反复进行，长期声誉比短期获利更重要

既济 → 未济 → 循环

博弈不止一轮。"以未济结尾"= 这场博弈永远没有终局。

一句话：博弈论是用数学语言写的易经，易经是用符号语言写的博弈论。冯·诺伊曼在1944年用数学公式化了文王在前1050年用直觉捕捉到的东西——在一组力量配置下，系统的稳定态是什么，以及它会怎么演化。64卦 = 64种纳什均衡。变爻 = 均衡转移。文王卦序 = 均衡之间的演化路径。
